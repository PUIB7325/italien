# Dolce Vita - Learn Italian

Une application moderne pour apprendre l'italien, conçue avec une esthétique premium et une flexibilité pédagogique.

## Démarrage Rapide

1.  Installer les dépendances :

    ```bash
    npm install
    ```

2.  Lancer le serveur de développement :
    ```bash
    npm run dev
    ```

## Fonctionnalités

- **Design Premium** : Interface sombre moderne avec accents vibrants et glassmorphism.
- **Choix du format** :
  - _Micro-leçons_ : 5-10 min par jour.
  - _Modules thématiques_ : Voyage, Business, etc.
  - _Cours structurés_ : Approche académique.
